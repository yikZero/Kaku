## Zero-effort config migration

Your existing `~/.wezterm.lua` or `~/.config/wezterm/wezterm.lua` can be reused by Kaku as-is.

## What's different

Kaku adds AI-focused configuration options on top of WezTerm and ships tuned defaults for macOS. See [Lua Configuration](/Kaku/en/docs/config/lua/) for the Kaku-specific options.
