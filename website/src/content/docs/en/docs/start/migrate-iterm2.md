description: A guide for iTerm2 users switching to Kaku

## Familiar shortcuts

Kaku keeps the iTerm2 keybindings you already have muscle memory for: `⌘T` new tab, `⌘W` close, `⌘D` split.

## Configuration

iTerm2's plist profiles cannot be imported directly, but Kaku's defaults are tuned to feel good out of the box — most users do not need to migrate anything. When you do want to tweak things, configuration is done in Lua; see [Lua Configuration](/Kaku/en/docs/config/lua/).
