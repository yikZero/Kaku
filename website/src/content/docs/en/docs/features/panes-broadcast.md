## Splitting

- `⌘D` — split vertically
- `⌘⇧D` — split horizontally
- `⌘⌥ + arrows` — move between panes

## Broadcast input

Send the same keystrokes to multiple panes at once. See [Keybindings](/Kaku/en/docs/config/keybindings/) for the broadcast shortcuts.
