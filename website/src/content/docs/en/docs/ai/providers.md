## Built-in provider presets

Run `kaku ai` to open the AI settings panel. The Provider dropdown lists built-in presets — picking one auto-fills the Base URL and populates the model dropdown.

## Custom provider

Pick **Custom** to manually enter a Base URL and API Key for any OpenAI-compatible endpoint.
